# Testing with Mocha ☕️

## 🤷‍ How to use this repo
1. Fork it
2. Clone it
```
git clone https://github.com/ap4gh/testing_with_mocha.git
```
3. Install dependencies
```
npm install
```
4. Run tests
```
npm run tests
```
5. To practice from start use `start` branch
```
git checkout start
```
